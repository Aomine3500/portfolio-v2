# Ahmed Znouda - Modern Portfolio v2

This is a modernized version of Ahmed Znouda's portfolio, built with React, TypeScript, Tailwind CSS, and Framer Motion.

## Features

- **Responsiveness**: Fully responsive design across all devices.
- **Bilingual Support**: Toggle between French and English.
- **Dark Mode**: Seamless toggle between light and dark themes.
- **Interactivity**: Smooth scroll-triggered animations and interactive elements using Framer Motion.
- **Modern UI**: Clean, professional design with a focus on UI/UX best practices.

## Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Environment Variables

Set the `GEMINI_API_KEY` in `.env.local` for AI-related features.
